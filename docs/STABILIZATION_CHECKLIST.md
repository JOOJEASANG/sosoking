# 소소킹 배포 점검표

## 배포 전

### 코드와 설정

- [ ] `npm run check` 통과
- [ ] 모든 로컬 `import`와 `require` 경로 존재
- [ ] `public/index.html`의 CSS·JS·이미지 경로 존재
- [ ] `app-module-registry.js` 등록 파일 존재
- [ ] `functions-main-v2.js`에 같은 함수 이름의 중복 구현 없음
- [ ] `firebase.json`, Rules, Indexes JSON 구문 정상
- [ ] `GEMINI_API_KEY`가 Firebase Secret Manager에 등록됨

### 배포 범위

Functions 또는 Rules가 변경되면 전체 범위를 배포합니다.

```bash
firebase deploy --only firestore:rules,firestore:indexes,functions,hosting --project sosoking-481e6
```

화면 파일만 변경된 경우에만 Hosting 단독 배포를 사용합니다.

```bash
firebase deploy --only hosting --project sosoking-481e6
```

## 배포 후 화면

### 공통

- [ ] 첫 접속 시 빈 화면 없이 앱이 열림
- [ ] 콘솔에 모듈 404 또는 구문 오류가 없음
- [ ] 새로고침과 해시 URL 직접 접속이 정상
- [ ] 라이트·다크 테마 전환 정상
- [ ] 서비스워커 갱신 후 이전 파일이 계속 남지 않음

### 홈

- [ ] 판결·상담·토론·드립 4개 공간이 표시됨
- [ ] 각 공간 버튼이 해당 글쓰기 화면으로 이동
- [ ] 오늘의 베스트와 인기 참여글 표시
- [ ] 인기 댓글이 게시글 상세로 이동

### 피드

- [ ] 목록·정렬·검색·페이지 이동 정상
- [ ] 게시글 유형 표시가 현재 4개 공간과 일치
- [ ] 게시글 클릭 시 상세 화면 정상
- [ ] 비공개 또는 숨김 게시글이 일반 사용자에게 노출되지 않음

### 글쓰기

- [ ] 판결 작성과 선택지 저장 정상
- [ ] 상담 작성과 상담 설정 저장 정상
- [ ] 토론 작성과 찬반 선택지 저장 정상
- [ ] 드립 주제 작성 정상
- [ ] 이미지 업로드·삭제·대기 상태 처리 정상
- [ ] 임시저장과 수정 화면 정상
- [ ] 모바일 입력창 확대·겹침 없음

### 상세와 참여

- [ ] 본문과 이미지 표시 정상
- [ ] 투표·반응·댓글·답글 정상
- [ ] 조회수 중복 제한 정상
- [ ] 작성자 수정·삭제 정상
- [ ] 비회원에게 로그인 안내 표시
- [ ] 기존 레거시 게시물도 최소한 읽을 수 있음

### AI

- [ ] 새 게시물에 유형에 맞는 AI 캐릭터가 선택됨
- [ ] 캐릭터별 말투와 관점이 다름
- [ ] 같은 고정 문장이 여러 유형에 반복되지 않음
- [ ] 관리자 테스트에서 기존 AI 댓글 교체 정상

- [ ] 하단 네비게이션이 콘텐츠를 가리지 않음

## 문제 발생 순서

1. 개발자도구 Console 확인
2. Network에서 실패한 JS·CSS·Functions 요청 확인
3. 서비스워커 등록 해제 후 새로고침
4. GitHub Actions의 배포 커밋 확인
5. Firebase Functions 로그 확인
6. Firestore Rules와 `admins/{uid}` 문서 확인
