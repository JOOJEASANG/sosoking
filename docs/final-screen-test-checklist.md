# 소소킹 최종 화면 테스트 체크리스트

이 문서는 배포 후 실제 브라우저와 모바일 기기에서 확인할 항목을 정리한 문서입니다.

## 1. 배포 상태 확인

- [ ] GitHub 최신 커밋이 정상 배포 표시인지 확인
- [ ] Firebase Hosting 배포가 성공 상태인지 확인
- [ ] 브라우저 새로고침 후 오래된 화면이 남지 않는지 확인
- [ ] 모바일에서 앱 설치/PWA 캐시 후에도 최신 화면이 보이는지 확인

## 2. 관리자 화면 공통

경로: `/admin`

- [ ] 관리자 계정으로 접속 가능
- [ ] 일반 계정은 관리자 화면 접근 차단
- [ ] 상단에 `현재 시스템 빠른 점검` 패널 표시
- [ ] `확장 모듈 정상 로드` 문구 표시
- [ ] 모바일에서 관리자 메뉴가 가로 스크롤로 표시
- [ ] 관리자 메뉴 클릭 시 화면이 깨지지 않음

## 3. 관리자 피드 관리

경로: `/admin` → `피드 관리`

- [ ] 최신 게시글 100개 표시
- [ ] 게시글 제목 클릭 시 상세 페이지 이동
- [ ] 숨김 버튼 클릭 시 게시글 숨김 처리
- [ ] 숨김 해제 버튼 클릭 시 공개 상태 복구
- [ ] 삭제 버튼 클릭 시 확인창 표시
- [ ] 삭제 시 서버 deep delete 경로 사용
- [ ] 삭제 후 목록에서 해당 게시글 제거
- [ ] 모바일에서 표가 화면 밖으로 터지지 않고 가로 스크롤됨

## 4. 관리자 신고 처리

경로: `/admin` → `신고 처리`

- [ ] 신고 처리 탭 표시
- [ ] 최근 신고 수 표시
- [ ] 처리 대기 수 표시
- [ ] 신고 목록 표시
- [ ] 신고 대상 게시글 제목/ID 표시
- [ ] 숨김 버튼으로 신고 대상 게시글 숨김 처리
- [ ] 삭제 버튼으로 신고 대상 게시글과 하위 데이터 삭제
- [ ] 완료 버튼으로 신고만 처리완료 표시
- [ ] 처리 후 목록이 새로고침됨
- [ ] 모바일에서 신고 표가 가로 스크롤됨

## 5. 소소공화국 허브

경로: `/republic`

- [ ] 공화국 허브 정상 표시
- [ ] 권력 사다리 표시
- [ ] 내 정치 상태 표시
- [ ] 대통령/대선/정당/기관 흐름 안내 표시
- [ ] 오늘 정치 루틴 패널 표시
- [ ] 모바일에서 카드가 1열로 안정 표시

## 6. 정치배틀

경로: `/battle`

- [ ] 오늘의 정치배틀 표시
- [ ] 투표 버튼 동작
- [ ] 토론 입력창 표시
- [ ] `토론 멘트 추천` 버튼 표시
- [ ] 추천 버튼 클릭 시 토론 입력창에 문구 채움
- [ ] `토론력 평가` 버튼 표시
- [ ] 토론 멘트 입력 후 설득력/현실성/방어력/대중성/말실수 위험도 점수 표시
- [ ] `상대 반박 예상` 버튼 표시
- [ ] 상대 반박 카드와 재반박 문장 버튼 표시
- [ ] 의견 제출 후 내 댓글 아래 `내 토론 결과` 카드 표시
- [ ] 80점 이상 댓글에 `명연설` 뱃지 표시
- [ ] 90점 이상 댓글에 `전설의 연설` 뱃지 표시
- [ ] 새로고침 후 같은 기기에서 저장된 토론 결과 카드 복원
- [ ] 공화국/대선/정당 연결 안내 표시
- [ ] 모바일에서 배틀 카드가 깨지지 않음

## 7. 정당 화면

경로: `/parties`

- [ ] 3개 정당 표시
- [ ] 정당 가입/탈당 흐름 정상
- [ ] 정당 정치력/당원 수 표시
- [ ] 정당 상세 이동 정상
- [ ] 모바일에서 카드와 버튼이 겹치지 않음

## 8. 대선 화면

경로: `/election`

- [ ] 대선 후보 목록 표시
- [ ] 후보 공약 영역 표시
- [ ] `공약 추천` 버튼 표시
- [ ] 추천 버튼 클릭 시 공약 입력창 채움
- [ ] 투표 버튼 동작
- [ ] 대통령 포고령 입력 영역 표시
- [ ] `포고령 추천` 버튼 표시
- [ ] 대통령에게 질문/대정부 질문 섹션이 표시되지 않음
- [ ] 보상 안내 문구가 `+10P 확정`처럼 오해되지 않음
- [ ] 탄핵 관련 문구가 `탄핵소추` 흐름으로 표시
- [ ] 모바일에서 후보 카드가 1열로 정렬

## 9. 국회 화면

경로: `/congress`

- [ ] 국회 페이지 표시
- [ ] 법안 목록 표시
- [ ] 법안 표결 버튼 표시
- [ ] 탄핵 관련 흐름 안내 표시
- [ ] 모바일에서 의석/법안 카드가 1열로 정렬

## 10. 헌법재판소 화면

경로: `/constitutional-court`

- [ ] 헌재 페이지 표시
- [ ] 탄핵심판/위헌심사 정보 표시
- [ ] `AI 재판관 의견 보기/생성` 버튼 표시
- [ ] 버튼 클릭 시 기존 의견 확인 또는 생성 동작
- [ ] 모바일에서 통계 카드가 1열로 정렬

## 11. 시민광장

경로: `/feed`

- [ ] 더보기 메뉴에서 `시민광장`으로 표시
- [ ] 시민광장 상단 안내 배너가 `공화국 여론` 성격으로 표시
- [ ] 정당 홍보, 공약 토론, 정치 의견, 헌재기록 문구 표시
- [ ] `시민발언 +20P`, `시민토론 +10P` 문구 표시
- [ ] 기존 `/feed` 경로와 게시글 목록은 정상 유지

## 12. 이용안내·약관·개인정보

경로: `/guide`, `/terms`, `/privacy`

- [ ] 이용안내가 소소공화국 게임 루프 중심으로 표시
- [ ] 약관에 정치배틀, 정당, 대선, 국회, 헌재, 포고령, 탄핵 내용 반영
- [ ] 개인정보처리방침에 현재 수집/이용 항목 반영
- [ ] 모바일에서 표와 목차가 가로 스크롤 또는 1열로 안정 표시

## 13. 소소신문

경로: `/news`

- [ ] 소소신문 페이지 표시
- [ ] 오늘의 기사 표시
- [ ] 정치배틀/정당/대통령/소소신문 중심 정보 표시
- [ ] 대통령 질의응답 하이라이트가 표시되지 않음
- [ ] 이번 주 국정 위기 국민투표가 표시되지 않음

## 14. SEO·공유 링크

- [ ] `/sitemap.xml` 접속 가능
- [ ] sitemap에 `/republic`, `/battle`, `/parties`, `/election`, `/congress`, `/constitutional-court`, `/news`, `/guide` 포함
- [ ] `robots.txt`에서 sitemap 주소 표시
- [ ] 카카오톡/문자 공유 시 제목이 `소소킹 - 가상 정치 공화국` 계열로 표시
- [ ] 공유 설명이 정당·대선·국회·헌재 구조로 표시

## 15. 모바일 실기기 확인

- [ ] Android Chrome에서 홈 진입 정상
- [ ] Android Chrome에서 관리자 화면 스크롤 정상
- [ ] Android Chrome에서 대선/배틀 버튼 겹침 없음
- [ ] iPhone Safari에서 관리자 메뉴 가로 스크롤 정상
- [ ] PWA 설치 후 재실행 시 최신 화면 표시

## 16. 오류 발생 시 우선순위

1. 관리자 상단 `확장 모듈 정상 로드` 여부 확인
2. GitHub 최신 커밋 배포 상태 확인
3. Firebase Functions 로그 확인
4. 브라우저 개발자도구 콘솔 오류 확인
5. 문제가 나온 보강 모듈만 분리 점검

## 17. 통과 기준

- [ ] 관리자 주요 탭 접속 가능
- [ ] 일반 사용자 핵심 루프 접속 가능
- [ ] 피드 삭제/신고 처리 안전 동작
- [ ] 시민광장이 공화국 여론 게시판처럼 표시됨
- [ ] 모바일에서 주요 카드와 표가 터지지 않음
- [ ] 대통령 질문/주간 위기 투표 보조 섹션이 표시되지 않음
- [ ] 정치배틀 토론력 평가가 표시됨
- [ ] 정치배틀 상대 반박 예상이 표시됨
- [ ] 댓글 제출 후 토론 결과 카드가 표시됨
- [ ] 고득점 토론 댓글에 명연설 뱃지가 표시됨
- [ ] sitemap/robots/share meta 정상
- [ ] 최신 커밋 최종 배포 성공
