## 이번 안정화에서 정리한 영역

### 1. 소소공화국 구조 정리

- 공화국 허브 권력 사다리 보강
- 홈 화면 소소공화국 진입 카드 보강
- 모바일 하단 메뉴에 대선 탭 우선 노출
- 정치배틀, 정당, 대선, 국회, 헌재 흐름 연결 안내 보강

### 2. 대선·대통령 기능 보강

- 대선 후보 공약 추천 버튼 추가
- 대통령 포고령 추천 버튼 추가
- 대선 보상 안내 문구를 서버 보상 로직과 어긋나지 않게 수정
- 불신임 표현을 탄핵소추 흐름으로 정리

### 3. 국회·헌법재판소 흐름 보강

- 국회·헌재 권력 견제 흐름 안내 추가
- 헌재 AI 재판관 의견 버튼을 `보기/생성`으로 보정
- sitemap에 `/congress`, `/constitutional-court` 공개 URL 포함

### 4. 약관·개인정보·이용안내 정리

- 이용약관을 소소공화국, 정치배틀, 정당, 대선, 국회, 헌재, 포고령, 탄핵 구조에 맞게 정리
- 개인정보처리방침을 현재 수집·이용 항목에 맞게 정리
- 이용안내를 정치게임 루프 중심으로 개편

### 5. 관리자 페이지 보강

- 관리자 현재 시스템 빠른 점검 패널 추가
- 관리자 문구에서 예전 `AI킹`, `판결소` 표현 보정
- 피드 삭제를 서버 `deleteFeedPostDeep` 기반으로 보강
- 신고 처리 탭 추가
- 신고 숨김, 삭제, 처리완료 기능 추가
- 확장 모듈 로드 상태 표시 추가

### 6. 모바일 반응형 보강

- 관리자 메뉴 모바일 가로 스크롤 처리
- 관리자 표, 신고 표, 약관 표 가로 스크롤 처리
- 공화국, 국회, 헌재, 대선, 배틀 카드 1열 반응형 보강
- 이용안내 목차와 카드 모바일 보강

### 7. SEO·공유·사이트맵 보강

- 기본 SEO 문구를 현재 정치게임 구조로 수정
- index.html OG/Twitter 공유 문구 수정
- sitemap에 현재 핵심 공개 페이지 추가
- sitemap XML 캐시 방지 보강

## 운영 확인 방법

### 관리자 화면

관리자 페이지 상단에서 다음 문구를 확인합니다.

```text
확장 모듈 정상 로드
```

오류가 있으면 해당 줄에 실패한 모듈 파일명이 표시됩니다.

### GitHub/Firebase 배포

커밋 직후 느낌표가 잠깐 보일 수 있습니다. 최종적으로 정상 배포 표시가 뜨면 문제 없는 상태로 봅니다.

### 캐시

서비스워커 캐시 버전은 안정화 과정에서 계속 증가했습니다. 이 문서 작성 시점 이후 최신 변경은 `public/sw.js`의 `CACHE` 값을 기준으로 확인합니다.

## 주의할 점

- 과거에 되돌린 깨진 커밋은 재적용하지 않습니다.
- 보강 모듈은 기능별로 분리되어 있어 당장 통합 삭제하지 않습니다.
- 화면 이상이 생기면 먼저 관리자 상단의 확장 모듈 상태를 확인합니다.
- 게시글 삭제는 가능하면 서버의 deep delete 경로를 사용합니다.
- Functions를 직접 수정할 때는 `npm run check` 기준 문법 검사를 통과해야 합니다.
