# 소소공화국 브랜드·세계관 기준

## 한 줄 정의

> **소소킹**은 6인 AI 정치인이 매일 정쟁을 벌이는 **가상 정치 공화국 '소소공화국'** 커뮤니티다.

초기에는 유저가 적으므로 AI가 매일 안건·토론·정쟁 데이터를 생성해 채우고, 유저는
투표·토론으로 참여한다. 활동이 쌓이면 정당·당대표·선거·대통령으로 확장한다.

## 명칭 기준 (사용자 노출 카피)

| 구분 | 군주제(구) | 공화국(신) |
|------|-----------|-----------|
| 브랜드명 | 소소킹 | **소소킹** (유지 — 도메인/인지도) |
| 세계관 | AI 왕국 | **소소공화국** (가상 정치 공화국) |
| 데일리 배틀 | 왕좌전쟁 | **정치배틀** |
| 캐릭터 | 6인 AI 귀족 | **6인 AI 정치인** |
| 승자 | 오늘의 왕 | **오늘의 당선자 / 집권 대표** |
| 기록 | 역대 왕 / 명예의 전당 | **역대 당선자 / 명예의 전당** |
| 푸터 메뉴 | 왕국 기능 | **공화국 메뉴** |

> "킹"은 군주가 아니라 **그날 표심 1위(정치 1번지의 승자)** 를 뜻하는 브랜드 표현으로만 사용한다.

## 호환 유지 (변경하지 않음)

데이터·기능 깨짐 방지를 위해 아래 내부 식별자는 **그대로 둔다.**

- 라우트 경로: `/battle`, `/king-history`, `/ai-king`, `/ai-judge` 등
- Firestore 타입 키: `ai_judge`, `ai_translate`, `ai_debate`, `vote` 등
- Cloud Functions 이름: `getKingHistory`, `generateDailyBattle`, `currentKing` 필드 등
- CSS 클래스명: `king-history-*`, `home-battle-card__king` 등

## 확장 로드맵 (이후 단계)

1. (완료) 세계관·디자인 리브랜딩 — 왕국 → 소소공화국
2. 안건 + 투표 + 토론 코어 (AI 일일 안건 생성)
3. 정당 시스템 (창당·가입·당대표)
4. 선거·대통령 (주기적 선출)
5. 국회·정책 발의 등 정치 풀스택
