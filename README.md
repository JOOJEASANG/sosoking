# SOSOKING

소소한 추억이 모이는 대한민국 방송 아카이브입니다.

## 공개 화면

- 방송사·연대·장르 필터
- 프로그램 검색과 상세 보기
- 랜덤 추억과 연도별 탐색
- 추억 작성 UI
- 모바일 반응형 디자인
- 초기 샘플 데이터 24건

## 관리자 화면

`/admin.html`에서 프로그램 데이터를 추가·수정·삭제할 수 있습니다.

현재 관리 화면은 브라우저 `localStorage`를 이용한 검수용 버전입니다.

- 프로그램 기본정보 편집
- 공개/검수 중 상태 관리
- 공식 출처 URL 기록
- JSON 일괄 가져오기
- JSON 백업 내보내기

## Firebase

프로젝트: `sosoking-481e6`

`firestore.rules`에는 다음 정책이 포함됩니다.

- 공개 프로그램만 일반 사용자에게 노출
- 일반 사용자의 추억 글은 `pending` 상태로만 등록
- 승인된 추억 글만 공개
- 프로그램 수정과 추억 승인·삭제는 관리자만 가능

관리자 권한은 Firebase Authentication 사용자에게 custom claim `admin: true`를 부여하는 방식으로 연결할 예정입니다.

## 배포 전 확인

1. 초기 프로그램 정보의 방영 기간과 출연진을 공식 자료로 재검수
2. 공식 출처 URL 입력
3. Firebase Authentication과 관리자 custom claim 연결
4. 공개 화면의 추억 작성 폼을 Firestore에 연결
5. 개인정보처리방침·운영정책·신고 기능 추가

방송 화면 캡처와 포스터는 권리 확인 없이 저장하지 않습니다.
