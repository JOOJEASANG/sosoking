사용자 화면 확인

홈, 게임, 피드, 인기, 글쓰기 화면이 같은 스타일로 보이는지 확인하세요.

소소피드사진, 글, 투표, 댓글 중심의 전체 콘텐츠 흐름을 관리합니다.
게임 메뉴밸런스 아레나, 사진 제목 배틀, 센스 퀴즈 등 놀이 진입 흐름을 확인합니다.
신고/의견부적절한 콘텐츠 신고와 사용자 개선 의견을 우선 확인합니다.
문서/정책이용안내, 약관, 개인정보, AI 안내가 현재 서비스 방향과 맞는지 점검합니다.

  • 게임 메뉴가 글쓰기 진입 메뉴로 자연스럽게 작동하는지 확인
  • 신고/의견접수/안내/약관 링크가 정상 동작하는지 확인
